 # Clima

Proyecto simple que muestra el clima utilizando React y Vite.

## Descripción

Este proyecto consume una API para obtener datos meteorológicos y los muestra de forma dinámica con React.

## Tecnologías

- React
- Vite
- TypeScript 
- Axios 

## Instalación

```bash
npm install
npm run dev
